--- FIX 2: Strip markdown links from Groq/LLM output ---

The 8b model was outputting markdown links like [text](url) in the 
generated text. Even though we're switching to Mistral, add a 
sanitization step to be safe.

In app/api/context/route.ts, after parsing the JSON response,
before returning, sanitize both strings:

function stripMarkdown(text: string): string {
  // Remove markdown links: [text](url) → text
  return text
    .replace(/\[([^\]]+)\]\([^)]+\)/g, '$1')
    // Remove bare URLs
    .replace(/https?:\/\/\S+/g, '')
    // Remove leftover parentheses from URL removal
    .replace(/\(\s*\)/g, '')
    .trim();
}

Apply to both innerWorld and theMoment before returning.
